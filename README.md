# 특수학교 교사교육과정 플랫폼 (Teacher Curriculum Platform)

## 개요
특수학교 교사를 위한 교육과정 관리 및 공유 플랫폼

## 주요 기능 (예정)
- [ ] 교육과정 설계 및 관리
- [ ] 수업 자료 공유 및 아카이빙
- [ ] 교사 간 협업 도구
- [ ] 학생 개별화 교육계획(IEP) 연동 등

## 기술 스택
- **Frontend**: Next.js / Vite (React)
- **Styling**: Vanilla CSS (Premium Design)
- **Backend**: FastAPI / Node.js
